# 🎨 UI Guide

## Style Reference

Inspired by:
- Discord
- Steam
- Windows 11 Settings

---

## 🎨 Color Palette

- Background: #0B0D10
- Sidebar: #0F1218
- Cards: #151A21
- Accent: #3B82F6
- Text Primary: #FFFFFF
- Text Secondary: #A0A0A0

---

## 🔤 Typography

- Font: Inter or Segoe UI Variable
- Titles: Bold
- Body: Regular
- Numbers: Semi-bold

---

## 📐 Layout Rules

- Sidebar always visible
- Main content area scrollable
- 8px spacing system
- 12–16px border radius

---

## 🧩 UI Components

### Cards
- Rounded
- Hover glow
- Slight shadow

### Buttons
- Rounded pill shape
- Accent color
- Hover scale effect

### Sidebar
- Icons + labels
- Active indicator bar
- Smooth transitions

---

## 🖥 Pages

### Dashboard
- CPU usage
- RAM usage
- Storage
- Optimization score
- Quick action button

### Optimize
- List of tweaks
- risk level
- toggle system

### Cleanup
- temp files
- recycle bin
- cache cleaning

### Startup
- list apps
- enable/disable

---

## 🎯 Goal

Make the app feel like a modern gaming utility, not a Windows control panel.
