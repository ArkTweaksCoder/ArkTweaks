# 🧠 Ark Tweaks

Ark Tweaks is a Windows optimization tool built for gamers and performance-focused users.

It provides safe, transparent, and reversible system optimizations including:
- cleanup tools
- startup management
- power plan switching
- performance monitoring
- system tweaks with full explanations

---

## 🚀 Goal

To build a trusted, modern Windows utility that improves performance without breaking system stability.

---

## 🧱 Tech Stack

- C#
- WPF (.NET 8+)
- MVVM architecture
- Windows native APIs

---

## ⚙️ Features (MVP)

- System Cleanup
- Startup Manager
- Power Plan Switcher
- System Info Dashboard
- Restore Point Creation

---

## 🧠 Core Principles

- Every tweak is reversible
- No unsafe registry hacks by default
- Transparency over “fake FPS boost” claims
- User always has control
